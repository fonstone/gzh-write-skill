# 四层自检金字塔

## 设计理念

参考软件工程测试金字塔：从最基础的硬性规则(自动扫描) → 模式匹配(可量化) → 深度审查(需判断) → 活人感终审(主观但可指导)。每层有明确通过标准和修复指引。

---

## L1 硬性规则检查（自动扫描层）— 必须 0 容忍

### L1-1 禁用词扫描

全文搜索以下词汇，**出现即必须替换**：

| 禁用词 | 替换建议 |
|--------|---------|
| 说白了 | 坦率的讲 / 其实就是 |
| 意味着什么 / 这意味着 | 那结果会怎样呢 / 所以呢 |
| 本质上 | 说到底 / 其实 |
| 换句话说 | 你想想看 / 也就是说 |
| 不可否认 | 直接删掉，换正面陈述 |
| 综上所述 / 总的来说 | 具体的回扣句 |
| 首先...其次...最后 | 自然转场词(说到这个/回到xxx/顺着上面) |
| 值得注意的是 / 不难发现 | 删掉，直接说 |
| 作为一个 / 让我们 | 删掉，换第一人称口语 |
| 众所周知 / 事实上 / 显而易见 | 删掉 |

### L1-2 禁用标点扫描

| 禁用标点 | 替换 |
|----------|------|
| 冒号 `：` | 逗号 |
| 破折号 `——` | 逗号或句号 |
| 双引号 `""` `""` | `「」` 或直接不加引号 |

### L1-3 结构性套话扫描

- `让我们来看看...` / `接下来让我们...` → 删掉
- `在当今...的时代` / `随着...的发展` → 删掉，从具体事件切入
- 连续 bullet point 超过 3 个 → 改散文叙述
- 大段加粗超过 2 行 → 砍到关键词只加粗

### L1-4 工具名具体化检查

- 禁用："AI工具"、"某个模型"、"相关技术"、"大模型"
- 必须用具体名：Claude Code、Seedance 2.0、Deepresearch、Clawbot、豆包、Kimi 等

### L1-5 假设性例子检查

- 禁用："比如有一次..."/"我见过一个用户..."/"假设你是一个..."——这些是 AI 编造场景的标志性表达
- 替换为真实经历："就像我今天正在搞的..."/用户提供的真实案例/标注编辑锚点让用户补
- 如果你没有真实细节，不如写"我自己还没试过，但想想就觉得xxx"

### L1-6 AI 角色边界检查

- 确认**第一手观察/核心创意角度/真实情绪**不是 AI 生成的（必须来自用户或编辑锚点待补）
- 确认没有 AI 越界编写了它不该碰的内容（如编造个人经历）
- 如果某个段落的"我"没有真实经历支撑，标注编辑锚点以待用户补充

**通过标准**：L1-1~L1-6 全部零命中  
**修复方式**：逐个替换；假设性例子删掉改标注编辑锚点；AI 越界段标注待用户补

## L2 风格一致性检查（模式匹配层）— 量化指标

### L2-1 开头检查

- [ ] 是否从**具体、当下的事件/场景**切入？(非宏大叙事)
- [ ] 第一句话是否让读者产生"然后呢？"冲动？
- [ ] 无教科书式开头(`在当今...`/`随着...`)

### L2-2 节奏与结构检查

- [ ] **长短句交替**：无连续 3 句以上长度相近(±5字)
- [ ] **断裂效果**：全文至少 3 处一句话独立成段
- [ ] **扣主线句**：偏离主线段落后必须有拉回句(频率高、可短)
- [ ] **疑问句节奏**：至少 2 处用疑问句作刹车/转向
- [ ] **无不必要小标题**：除非分条目方法论，否则靠口语化转场衔接
- [ ] **契诃夫之枪/回环呼应**：前文是否埋了后面会响的细节/意象？开头悬念型意象结尾回扣了吗？
- [ ] **前言后语情绪落差**：相邻段落间情绪/节奏是否有变化(非匀速推进)？

### L2-3 口语化检查

- [ ] **推荐口语词组** ≥ 8-10 个不同表达(见下表)
- [ ] **论述故意打破** ≥ 3 处(重复强调/中途打断/省略主语/刻意模糊)
- [ ] **自嘲/承认不足** ≥ 1 处
- [ ] **情绪标点** `。。。` `？？？` `= =` 至少出现一种

**推荐口语化词组表**（写作时主动植入，自检时计数）：

| 类别 | 高频词组(必中) |
|------|----------------|
| 转场过渡 | 坦率的讲、说真的、我是真的觉得、反正我觉得、怎么说呢、其实吧、你想想看、我跟你说、回到xxx这块、这块需要注意一下、顺着上面的再聊聊 |
| 表达判断 | 我有时候觉得、我一直觉得、这话听着有点刺耳但、不是说xxx不行而是说、我自己的感受是、我始终坚信、我觉得还是挺重要的 |
| 承认自嘲 | 说实话我也不确定、我自己也还在摸索、可能有些想法还不成熟、这个事儿我也踩过坑、愚钝如我、我说"理论上"是因为我自己还没完全跑通 |
| 情绪表达 | 这种感觉太爽了、我当时就愣住了、想想就觉得兴奋、我真的被震撼到了、搞得我现在还有点懵、太离谱了、太特么赤鸡了、给我一下子整不会了、一时间无语凝噎、鬼使神差的、你敢信？？？ |
| 拉近距离 | 很多朋友可能不知道、可能有小伙伴纳闷、你如果关注这个领域的话、大家也都知道 |
| 口头禅 | 这玩意、不是哥们、我寻思了一下我没寻思明白、有个屁的xxx、真的就是一声叹息、这尼玛就是、太牛逼了、比较骚的事 |

### L2-4 标点禁令二次确认

- [ ] 全文**零冒号、零破折号、零双引号**(AI 修改易重新引入)

**通过标准**：L2-1 全通过，L2-2 ≥ 5/7，L2-3 ≥ 3/4，L2-4 通过  
**修复方式**：逐段改写，重点修"读起来像写报告"的段落。回环呼应缺失 → 在关键位置补一个意象/句子的前文铺垫和后文回扣

## L3 内容质量检查（深度审查层）— 需人工判断

### L3-1 观点支撑检查

- [ ] 每个核心观点是否有**具体人/场景/细节/数据**支撑？
- [ ] 无空泛观点只有论断无例证？

### L3-2 知识输出方式检查

- [ ] 知识点是否**"聊着聊着顺手掏出来"**(非"下面我来科普")？
- [ ] 无"下面我来介绍"/"首先需要了解"教科书式科普？
- [ ] 引用(论文/书籍/历史)是否像"聊着聊着想起来的"自然融入？

### L3-3 文化升维检查

- [ ] 至少 1 处从具体事件连接到**更大文化/哲学/历史参照物**？
- [ ] 连接是否自然(像"聊着聊着想到了"而非硬凑)？

### L3-4 对立面与同理心检查

- [ ] 讲核心观点时，是否先**站在对方角度把处境具体化**？
- [ ] 是否**承认对方处境合理**("我非常理解这种感觉")，再给出不同视角？

### L3-5 文章类型专项检查(按原型)

| 原型 | 专项检查点 |
|------|------------|
| 调查实验型 | 有"亲自下场"叙事感？过程有层层递进发现？起点是读者能代入的具体困境/好奇(非抽象命题)？ |
| 产品体验型 | 有真实使用场景？有自然对比？是"跟我一起玩"而非评测报告？ |
| 现象解读型 | 有"观察→好奇→研究→升维"推进？升维自然(像聊着聊着想起来的)而非硬凑？ |
| 工具分享型 | 有个人故事铺垫(非直接"推荐一个好用的xxx")？效果展示让人"卧槽"？ |
| 方法论型 | 每节落到可执行行动(读完能立刻动手)？**坦诚说明学习成本/失败点**(不只画饼)？开头/结尾有谦逊铺垫？ |

### L3-6 逐一展示 + 升番逻辑检查

- [ ] 多产品/案例比较时，是否用**逐一展示+吐槽点评**(非一次性罗列结论)？
- [ ] 是否遵循**升番逻辑**：弱→强→最炸，制造"还能往上翻"惊喜？
- [ ] **排列顺序决定情绪曲线**：最弱的放前面、最炸的留最后，中间有"我以为到顶了结果还能往上翻"的惊喜感？
- [ ] 每个展示是否都带一句点评/吐槽(而不是平铺陈述)？

### L3-7 案例人物公正性检查

- [ ] 当用真实人物做案例时，是否展示了完整的故事弧(非只截取有利于论点的片段)？
- [ ] 是否避免了片面截取(让懂行的读者觉得在偷换概念)？

### L3-8 契诃夫之枪完成度检查

- [ ] 前文埋下的钩子/意象/细节，在文后是否得到了回扣/呼应？
- [ ] 如果开头有悬念型意象，结尾是否回扣了？(回扣不是重复，是视角变化后的重现)

**通过标准**：L3-1/L3-2 必须全通过，L3-3~L3-8 按类型至少通过相关项(不适用则跳过)  
**修复方式**：重审不通过段落，补案例/改写知识输出/调整升维自然度。案例公正性问题→补充完整故事弧或调整措辞。升番排列不对→重新排序

## L4 活人感终审（最终人格层）— 核心质检

**单一核心问题**：

> "读完这篇文章，我感觉是一个**有见识的普通人在认真跟我聊一件打动他的事**，还是一个 **AI 在给我输出信息**？"

### 具体感知维度

| 维度 | 判断标准 | 失败表现 |
|------|----------|----------|
| **L4-1 温度感** | 情绪是体感记忆("我当时就愣住了"/"鼻子一酸")而非知识性描述("我感到震撼")；人物有体温 | 全文情绪平铺、人物像纸片人 |
| **L4-2 独特性** | 有"只有这人能写出的角度"，非换个博主也能写同样东西 | 观点泛泛、结构模板化 |
| **L4-3 姿态** | "有见识的普通人聊打动他的事"，无"导师教学生"/"品牌做营销"感 | 居高临下、过度推销、过度客观 |
| **L4-4 心流** | 通读无注意力断点，无需回头顺逻辑 | 有卡顿处 = 节奏问题需修 |

**通过标准**：L4-1~L4-4 整体感"像真人写的"，任何一项"AI味重"即返工  

**逐维度修复指引**：

| 维度 | 失败表现 | 修复方法 |
|------|----------|---------|
| **L4-1 温度感** | 情绪平铺、人物纸片人、体感记忆被知识性描述替代 | 把"我感到震撼"改成"我当时就愣住了"/"鼻子一酸"。添加感官细节(台灯的颜色、外卖的味道、手指的温度)。检查每个情绪表达：这是体感记忆还是知识性描述？ |
| **L4-2 独特性** | 观点泛泛、换个博主也能写出同样的东西、像行业报告 | 找到文中"10个不同的人都会这么说"的句子，全部改或删。加入只有这个账号/人设才会有的判断角度。如果整篇没有"只有他能写出来"的观点，回到核心论点重新切角度 |
| **L4-3 姿态** | 居高临下、过度推销、过度客观、像在"教"读者 | 逐段检查语气：是在"分享"还是在"教导"？把"你需要注意"改成"我自己也在摸索"。加上"说实话我也不确定""我自己也还在犯错"。添加对立面理解段落 |
| **L4-4 心流** | 某处注意力断掉、需要回头顺逻辑、段落间衔接生硬 | 找到断点位置，检查：是不是偏离主线太远没扣回来？是不是段落间逻辑跳跃太大？是不是信息密度突然太高或太低？补一句扣主线句或加一句低密度过渡

## 附录：Humanizer 五维评分参考（适配自 Humanizer-zh）

作为 L4 活人感终审的补充量化工具，对改写后的文本按以下五维 1-10 分评分：

| 维度 | 核心问题 | 1-3 分（AI 味重） | 4-6 分（可接受） | 7-10 分（人类感） |
|------|---------|-------------------|-----------------|-----------------|
| **直接度** | 是否直接陈述事实、圈定重点？ | 拐弯抹角，过度铺垫 | 偶有冗余但整体直接 | 一针见血，开门见山 |
| **句式变化度** | 句子结构和长度是否变化丰富？ | 单调呆板，机械重复 | 有变化但偶有重复 | 错落有致，节奏感强 |
| **文本松弛度** | 是否保留自然松弛的语言节奏和呼吸感？ | 紧绷僵硬，过于刻板 | 部分段落松弛 | 松弛有度，恰到好处 |
| **真实感与温度** | 文本是否有真实的人情味和具体的画面感？ | 空洞虚假，机械僵硬 | 偶有具体场景细节 | 有血有肉，真诚自然 |
| **冗余度** | 是否有可删除的冗余废话或重复总结？ | 废话连篇，反复倒浆糊 | 少量冗余但可接受 | 干净利落，无一字多余 |

**评分规则**：
- 通读全文后给每个维度打分（1-10 分整数）
- 五维加总得到总分（满分 50 分）
- **45-50 分**：极度自然，几乎看不出 AI 痕迹
- **35-44 分**：良好，但还有小幅改进空间
- **35 分以下**：需要进一步修改和提升

**使用时机**：L4 活人感终审时，如果感觉"整体还行但说不上哪里不好"，用这五个维度做结构化诊断，定位最低的 1-2 个维度定向修复。不要求每次都跑这个评分，只作为 L4 量化补充。```
